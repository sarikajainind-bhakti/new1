# FryCuisine Website – Complete Color & Font Style Guide

## 1. Overall Design Style

The website uses a premium food-magazine inspired aesthetic with:

* Dark luxury hero sections
* Cream / off-white content background
* Elegant serif typography
* Warm food-focused accent colors
* Editorial blog layout style
* Minimal but classy UI

The overall mood is:
**Luxury + Food Blog + Modern Editorial + Premium Cuisine Branding**

# 3. Font Guide (Most Important)

The site mainly uses elegant serif fonts mixed with readable clean body fonts.

## A. Logo Font – “frycuisine”

### Closest Font:

* Playfair Display Bold
  OR
* Cormorant Garamond Bold

### Style:

* Serif
* Luxury editorial style
* High contrast thick/thin strokes

### Logo Color:

* White (#FFFFFF)

### Placement:

* Top left in navbar
* Footer center branding

---

## B. Main Headings Font

### Used In:

* “TRENDING FOOD RIGHT NOW”
* “Explore Trending, Global & Healthy Food”
* Recipe section titles

### Closest Font:

* Cinzel Decorative
  OR
* Cormorant SC

### Font Weight:

* 500–700

### Letter Style:

* Uppercase
* Wide spacing
* Editorial magazine look

### Heading Color:

* #1A1A1A
  or
* #F8F5F0 on dark sections

### Recommended Size:

| Device               | Font Size |
| -------------------- | --------- |
| Desktop Hero Heading | 54px–64px |
| Section Heading      | 42px–48px |
| Mobile Heading       | 28px–36px |

---

## C. Body Text Font

### Used For:

* Recipe descriptions
* Blog summaries
* Footer text
* Small content

### Closest Font:

* Lora
  OR
* Libre Baskerville
  OR
* Open Sans (small UI text)

### Font Weight:

* 300–400

### Color:

* #555555

### Recommended Size:

| Content Type    | Size      |
| --------------- | --------- |
| Paragraph       | 16px–18px |
| Small Meta Text | 13px–14px |
| Footer Links    | 16px      |

This website design combines:

* Luxury restaurant branding
* Editorial food magazine style
* Pinterest-inspired recipe layout
* Modern minimal UI
* Warm premium food aesthetics

The strongest visual elements are:

1. Elegant serif typography
2. Warm cream backgrounds
3. Dark cinematic hero images
4. Minimal luxury spacing
5. Magazine-style content cards
6. Gold/beige premium accents
